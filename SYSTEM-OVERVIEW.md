# System Overview

This repository contains applied tools and protocols for structured decision-making in complex environments.

The system is based on the core research framework:

Human Judgment as Structural Constraint in AI-mediated Systems.

---

## Structural Layers

This system consists of three layers:

### 1. Core (Definition Layer)
Defines the fundamental concepts:
- Human Judgment  
- Decision Architecture  
- Structural Boundaries  

All core definitions are maintained in the canonical repository.

---

### 2. Protocol (Governance Layer)
Defines structural rules and standards:
- Responsibility allocation  
- Decision traceability  
- System consistency  

Located in the `/protocols` directory.

---

### 3. Tools (Execution Layer)
Provides practical methods for application:
- Decision clarification  
- Responsibility mapping  
- Judgment boundary identification  

Located in the `/tools` directory.

---

## Relationship

Core → defines  
Protocol → structures  
Tools → executes
